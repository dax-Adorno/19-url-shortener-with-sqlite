# 19 - URL Shortener with SQLite

Python CLI application that shortens URLs locally, stores them in SQLite, avoids duplicates, and retrieves the original URL by short code.

## Features

- Shortens long URLs into short local codes
- Stores URLs in SQLite
- Prevents duplicate entries
- Retrieves original URLs by code
- Lists all saved URLs
- Input validation

## Technologies

- Python 3
- SQLite
